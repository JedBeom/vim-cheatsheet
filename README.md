## Move

- `%`: move to the matching bracket.
- `CTRL-e`: Scroll the window downwards.

## Text objects

- `diw`: Delete Inside Word
- `ciw`: Change Inside Word (delete inside word and start INSERT mode)
- `yip`: Yank(copy) Inside Paragraph
